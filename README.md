A console-based minesweeper game!

## 🎮 How to play 🎮
### 1. Enter the following 💻
   - x and y board size (maximum 60)
   - Number of mines
### 2. Choose an action 🎲
   - [S] for show
   - [M] for mark
   - [R] for restart
   - [Q] for quit



💣 Your goal is to reveal all cells without opening the mined cell 💣
💥If you open a mined cell, then you lose! 💥

Good luck!
